# Red Store

## Overview

Red Store is a responsive e-commerce storefront for sneakers and sports shoes. The website showcases product categories, item details, and a dedicated cart and account experience using clean HTML, CSS, and JavaScript.

## Features

- Responsive homepage with featured products
- Product listing and details pages
- Shopping cart interface
- Account page layout
- Mobile-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Installation

1. Clone the repository.
2. Navigate to the project directory.
3. Open the HTML files in your web browser.
4. No additional dependencies are required.

## Usage

Open `Index.html` to view the homepage, browse products on `Products.html`, and explore product details on `Products Details.html`. Use `Cart.html` to preview the shopping cart layout and `Account.html` for account-related sections.
